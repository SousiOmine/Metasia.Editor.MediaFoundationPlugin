# Metasia.Editor.MediaFoundationPlugin

## ビルド

1. `MediaFoundationPlugin/local.props.template` を `MediaFoundationPlugin/local.props` にコピーする
2. `MetasiaApplicationPath` に `Metasia.Editor.Plugin.dll` と `Metasia.Core.dll` があるフォルダを指定する
3. `dotnet build MediaFoundationPlugin.sln`

## 対応環境

- Windows x64
- Windows arm64
